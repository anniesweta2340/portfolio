hybrid ai that tells you who really isn't following back

the problem

Instagram shows you a follower count but not who quietly drifted away. Existing tools either ask for your password, are buried in ads, or choke on large accounts. There's no native way to identify which personal accounts simply stopped following back — businesses and celebrities mixed in make the noise even worse.

what i built

The app parses your official instagram data export — no login, no third-party access, nothing leaves your device. A weighted pattern-recognition algorithm classifies every account as personal, business, celebrity, or school, then surfaces only the personal ones who don't follow back. An optional claude api key upgrades the analysis with real ai reasoning, but the core works entirely offline.

key features

  • Hybrid ai analysis — pattern recognition works offline; claude api is an optional upgrade, not a requirement
  • Unlimited scale — handles 1m+ followers across any number of json files with streaming progress updates
  • Smart filtering — automatically excludes businesses, celebrities, and schools so you only see accounts you actually know
  • Real-time progress — async file processing keeps the browser responsive throughout, no backend needed
  • Privacy-first — 100% in-browser; your data never leaves your device unless you explicitly opt into ai mode
  • Weighted scoring — confidence scores from username patterns, bio keywords, follower ratios, and verified-status signals

tech stack

React · claude sonnet 4 api · jszip · browser file api · async streams · github pages

challenges & solutions

Instagram splits large follower lists across multiple json files with no documented limit. I built a dynamic multi-file handler that merges unlimited files in memory with per-file progress — no cap on input size.

Most users don't have an anthropic api key. The answer was a hybrid architecture: pattern-matching covers 90%+ accuracy offline, and the api key unlocks an enhanced layer as an optional upgrade, not a gate.

Processing 100k+ accounts synchronously would freeze the browser. Chunked async processing with react state updates between batches keeps the ui live throughout, with classification running in microtask-sized passes.

Classifying personal vs. brand requires heuristics — there's no api access to instagram's own category data. The solution was a weighted scoring system across username patterns, follower-to-following ratio, bio keyword lists, and verified flag, all derivable from the export json alone.

impact

Processes datasets up to 1M+ followers in a single browser session, with 90%+ accuracy identifying personal vs. business accounts. Zero bytes of user data ever transmitted off-device.
